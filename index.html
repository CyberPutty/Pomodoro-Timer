How To Use This Timer

There are 3 timers you can use to start/stop, set the interval, and set the break time. Create a list of tasks in the Task List column and drag them into the Current Task column to add them to your timer. Your Pomodoro Timer will automatically check the tasks off at the end of each interval and tell you to go to break. If a task is running you will need to stop the timer to make changes.

According to Wikipedia there are six steps to the Pomodoro Technique:

  1. Decide on the tasks to be done.
  2. Set the pomodoro timer (traditionally to 25 minutes).
  3. Work on the task.
  4. End work when the timer rings and check off task.
  5. If you have fewer than four checkmarks, take a short break (3–5 minutes), then go to step 2.
  6. After four pomodoros, take a longer break (15–30 minutes), reset your checkmark count to zero, then go to step 1.
